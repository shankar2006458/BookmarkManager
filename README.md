# 📑 Smart Bookmark Manager

A simple and modern bookmark manager built with **HTML**, **CSS**, and **Vanilla JavaScript**. Easily organize and manage your favorite websites in one place.

## 🌐 Live Demo

https://shankar2006458.github.io/BookmarkManager/

## ✨ Features

* Add, edit, and delete bookmarks
* Organize bookmarks with categories
* Mark bookmarks as favorites
* Search bookmarks instantly
* Trash and restore deleted bookmarks
* Responsive design for desktop and mobile
* Data stored locally using LocalStorage

## 🛠️ Built With

* HTML5
* CSS3
* JavaScript (Vanilla)
* LocalStorage
